문제1 : 정보설계

1-1 Top메뉴
회원관련 : 회원가입 로그인 아이디찾기 비밀번호찾기 개인정보수정 로그아웃
개인상품관련 : 장바구니 적립금 1:1문의 주문내역 늘사는것

1-2 Main메뉴
main1(이벤트관련) : 이벤트소개 이벤트바로가기
main2(검색관련) : 검색 카테고리 카테고리상세검색
    카테고리세부메뉴
        과일/채소 : 쌈 샐러드 브로콜리 콩나물 버섯 시금치 부추 나물 양파 마늘 생각 파 파프리카 피망 고추 국산과일 수입과일
        수산물 : 새우 게 랍스터 오징어 낙지 문어 해산물 조개 수산가공품 김 미역 해조류 건어물
        육류 : 소고기 계란 닭 오리 양념육 돈까스 양고기
        견과 : 건과일 건과류
        쌀/잡곡 : 쌀 잡곡
        냉장/간편 : 국 탕 찌개 밑반찬 두부 어묵 햄 소시지 김치
side(추천관련) : 베스트 신상품

1)attr : HTML 의 속성(attribute)을 취급
            -> 속성은 HTML Element에 있는 정보
        2)prop : JavaScript의 프로퍼티(property)를 취급
            ->프로퍼티는 JavaScript에서 사용하는 정보
                (HTML의 정보와 일치 또는 불일치)

        $(선택자).attr(속성이름) : 속성값 가져오기
        $(선택자).attr(속성이름,값) : 속성에 값 넣어주기
        $(선택자).prop(속성이름[,값]) : 자바스크립트 입장에서의 속성
